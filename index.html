Coding Bootcamp project: Fractal explorer

My project is a Julia fractal explorer. I put it in the 'calm my nerves' category because I managed to make it quite smooth and visual. This project demonstrates the use of graphics, iteration, input and object orientation. As you can see in the cpu tab, I ended up using a multithreaded design for performance reasons, and eventually upgraded that to an OpenGL GPU version (as I had previous GLSL knowledge) so that the program can run smoothly. It includes a tutorial 'welcome' screen which you can open by pressing T, this lists the extensive controls. The basic controls are:

w/s: adjust real number.
a/d: adjust unreal number.
q/e: zoom in / out.

You can find it at: https://github.com/cameron-sim/adbs961
